# Exigences

Documents d'exigences BIM du projet, lus par le skill **qc-bep-reader** pour
extraire les règles de contrôle qualité appliquées par ChatQC.

| Fichier | Rôle |
|---|---|
| `Plan_Execution_BIM.docx` | BEP — Plan d'Exécution BIM (conventions, organisation, LOIN). |
| `Cahier_des_charges_BIM.docx` | Cahier des charges BIM (exigences contractuelles). |
| `Cahier_Participant_F11_DOP001.docx` | Cahier participant (exigences spécifiques au lot/disposition). |

> Pour auditer un modèle contre ces exigences, fournir le chemin du document
> dans le champ **BEP** du chatbox ChatQC (ou directement à `qc-orchestrator`).
