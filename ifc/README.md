# Modèles IFC — Snowdon Towers (SNW)

Maquettes IFC du projet de test, organisées par discipline. Servent de jeu de
données pour les audits ChatQC (à convertir en `.vim` puis charger dans VIM Flex).

| Fichier | Discipline |
|---|---|
| `SNW_A_ARC.ifc.ifc` | Architecture |
| `SNW_A_STR.ifc.ifc` | Structure |
| `SNW_A_CVC.ifc.ifc` | CVC (chauffage / ventilation / climatisation) |
| `SNW_A_ELE.ifc.ifc` | Électricité |
| `SNW_A_PLO.ifc.ifc` | Plomberie |
| `SNW_A_FAC.ifc.ifc` | Façade |
| `SNW_A_SIT.ifc.ifc` | Site / terrain |

> ⚠️ **Fichiers volumineux** (jusqu'à ~96 Mo). Proches de la limite GitHub de
> 100 Mo/fichier — envisager **Git LFS** si d'autres maquettes sont ajoutées.
> Le double suffixe `.ifc.ifc` provient de l'export d'origine ; conservé tel quel.
